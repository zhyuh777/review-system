> 来源：《电工电子技术基础》思考题与习题扫描配图（`extracted_questions/配图/`）  
> 共 **26** 张 PNG，对应 **26** 道含图习题。  
> 本文档包含：① 通用 AI 识别提示词；② **绘制友好** 的电位点描述模板；③ 逐图识图结果。  
> **绘制约定**：电源 type=`Battery`，符号为 **一长一短竖线 + 标注 ±**；检流计 type=`Galvanometer`（圆内一横线）；开路须显式标出断开的电位点对。

## 一、通用识别提示词

将下面内容作为 **System Prompt**；User 消息中附上图片路径或图片本身，并指定图号（如 `图1-22`）。

```text
你是一名电工电子技术教材配图分析助手。请从扫描版电路图 PNG 中识别电路结构，输出便于重绘（SVG/Canvas）的结构化描述。

【必须写清楚的内容】
1. 元件清单：逐个列出 type、参数、极性。所有直流/交流电源统一 type = "Battery"（含理想电压源、电池、标注 U_S 的电源）。
2. 电位点：为便于绘制，在电路中添加电位点 P0、P1、P2…（教材已有 a/b/c/d 时可复用字母，并在括号中对应 P 编号）。
3. 电位点之间的连接：用「P_m —[元件]— P_n」逐段描述；并联写「P_a 与 P_b 经 [R] 相连至 P_c」。
4. 开路/断开：若题目为开路、开断口、开口端子、开关断开等，必须单独列出 open_gaps，写明哪两个电位点之间无导线、或哪对端子悬空。
5. 电流参考方向：每个 I、I₁… 写清箭头方向，以及路径「从 P_x 经 … 到 P_y」。
6. 电压参考方向：U、U_x 等的 + 端、− 端分别接在哪个电位点。
7. 多子图 (a)(b)… 分别独立描述。

【元件 type 约定（绘制用）】
- Battery：直流/交流电源。符号 = **两条平行竖线，一长一短**（不是圆圈），**必须标注 + 与 −**，并写明 +、− 各接哪个电位点。交流源 u(t) 也用 Battery，注明 AC，仍画长短竖线 + ±。
- CurrentSource：电流源（圆圈内箭头）。
- Resistor / Capacitor / Inductor / Impedance：电阻、电容、电感、阻抗 Z。
- Galvanometer：检流计（P 表）。符号 = 圆形，中间一条横线（不是字母 P）。
- Ammeter：电流表 A（圆圈内字母 A）。
- Voltmeter：电压表 V（圆圈内字母 V）。
- Diode / BJT / Switch / Potentiometer：二极管 VD、三极管、开关 S、滑动变阻器。

【Battery 极性约定】
- 长线端旁标 **+**，短线端旁标 **−**（竖放时通常 + 在上或 + 在右，以原图为准）。
- JSON 中必须写 positive、negative 所接电位点编号。

【开路描述约定】
- open_gaps: [{ "between": ["P2", "P3"], "reason": "开断口，跨接电压 U" }]
- terminal_open: [{ "points": ["P7", "P8"], "reason": "右侧测量端子，未接负载" }]
- switch: [{ "between": ["Pa", "Pb"], "device": "S", "state_in_figure": "open" }]

【输出格式】
先中文说明（电位点表 + 电位点间连接 + 开路 + 电流），再附 JSON：

{
  "figure_id": "1-27",
  "potential_points": {
    "P0": "底公共地（左区）",
    "P1": "左区顶",
    "P2": "开断口左端（+）",
    "P3": "开断口右端（−）"
  },
  "connections": [
    {"from": "P0", "to": "P1", "through": [{"type": "Resistor", "value": "3Ω"}]},
    {"from": "P2", "to": "P3", "through": [], "open": true, "note": "开断，跨 U"}
  ],
  "components": [
    {"id": "Us", "type": "Battery", "value": "5V", "symbol": "long-short vertical lines", "positive": "P4", "negative": "P0", "polarity_marks": "+ at long line, - at short line"}
  ],
  "currents": [
    {"id": "I", "direction": "P1→P0", "path": "P1 — 3Ω — P0"}
  ],
  "open_gaps": [
    {"between": ["P2", "P3"], "reason": "顶部开断口"}
  ]
}
```

**User Prompt 模板：**

```text
请识别电路图：{文件名}
期望图号：{figure_id}
要求：
- 电源一律 type=Battery，符号=一长一短竖线，必须标 +/− 及所接电位点
- 检流计用 Galvanometer（圆内一横线）
- 列出电位点及电位点之间的连接
- 标出所有开路/断开位置
只输出结构化说明 + JSON。
```

### 第1章 电路的基本概念与基本定律

#### 图 1-22（`图1-22.png`）— 第3题

**电位点**

| 编号 | 含义 |
|------|------|
| P0 | 底边左角（U₁ 下端） |
| P1 | 顶边左角（U₁ 上端 = U₂ 左端） |
| P2 | 顶边右角（U₂ 右端 = U₃ 上端） |
| P3 | 底边右角（U₃ 下端） |

**元件清单**

| 编号 | type | 参数 | 连接电位点 | 说明 |
|------|------|------|------------|------|
| E1 | 一般二端元件 | U₁ | P1 — P0 | 电压参考 + 在 P1、− 在 P0 |
| E2 | 一般二端元件 | U₂ | P1 — P2 | 电压参考 − 在 P1、+ 在 P2 |
| E3 | 一般二端元件 | U₃ | P2 — P3 | 电压参考 + 在 P2、− 在 P3 |
| W1 | Wire | — | P0 — P3 | 底边导线 |

**电位点之间的连接**

- P1 —[E1/U₁]— P0 —[W1]— P3 —[E3/U₃]— P2 —[E2/U₂]— P1（闭合单回路，**无开路**）

**电流参考方向**

- **I₁**：P1 → P0，路径 P1 — U₁ — P0（向下）
- **I₂**：P1 → P2，路径 P1 — U₂ — P2（向右）
- **I₃**：P2 → P3，路径 P2 — U₃ — P3（向下）

**说明**  
三个一般元件构成矩形闭合回路；题目给定 I₁、I₂、I₃ 与 U₁、U₂、U₃ 数值，用于功率计算。

#### 图 1-23（`图1-23.png`）— 第4题

四个独立子电路 (a)–(d)，左侧均为 **开口端子对**（与右侧网络在竖直方向对齐，但左侧端口可视为外接负载端）。

**(a)**

| 电位点 | 含义 |
| Pa0 | 左端子（下） |
| Pa1 | 左端子（上）= 右网络底 |
| Pa2 | 右网络顶 |

**元件**：Battery **B_a** 8V，+ 接 Pa2，− 接 Pa1。

**连接**：Pa1 —[Battery 8V]— Pa2；Pa1、Pa2 分别连至左端子上下点。**左端子对本身与外部未短接**（二端口网络）。

**开路**：左开口端子 Pa0 与 Pa1 之间 **仅由端子定义**，外电路未画出 → `terminal_open: [Pa0, Pa1 为测量端口]`

**电流 I=2A**：参考 **流入** 左端口（从 Pa0/Pa1 侧进入右侧网络）。

**(b)**  
- Battery **B_b** 4V（+ 上 Pa2，− 下 Pa1）；**I=3A** 参考 **从 Battery 正极侧流出** 至左端子。

**(c)**  
- CurrentSource 1A↓（Pa2→Pa1）；左端口电压 **U=5V**（+ 在 Pa1 上端、− 在 Pa0 下端）。

**(d)**  
- CurrentSource 3A↓；**U=6V**（− 在上、+ 在下，与 (c) 反极性）。

#### 图 1-24（`图1-24.png`）— 第8题

四个子电路，均含测量仪表（内阻理想：A 为 0，V 为 ∞）。

**(a)**  
- **元件**：Battery 6V（+ 接节点 a 侧）、2 Ω 电阻、电流 **I**、伏特计 V（读数 8 V，+ 接 a、− 接 b）。  
- **连接**：节点 **a** 为左上；6 V 正极接 a；6 V 负极经 2 Ω 到节点 **c**；c 经标注 I 的支路到 **b**；a、b 间并接 V。  
- **电流 I**：参考方向沿 c→b 支路（图中在 2 Ω 下方支路标注）。

**(b)**  
- **元件**：未知 Battery U_S、2 Ω、Battery 5V（+ 在右）、Battery 2V（+ 在右）、5 Ω、电流表 A（1 A，+ 在上）、V（8 V，跨 a–b）。

- **电流**：A 读 1 A，方向从 a 经主回路流向 b（自上而下经过各串联元件）。

**(c)**

**(d)**  
- **元件**：A（1 A）、Battery 2V（+ 在 R 侧）、R、V（5 V，+ 接 a、− 接 b）。  
- **连接**：a → A → 2 V → R → b；V 并接 a–b。

**(a) 三并支路**

### 第2章 电阻电路分析

#### 图 2-18（`图2-18.png`）— 第1题

**(a) 菱形电阻网络**  
- **元件**：左竖 6 Ω，右竖 6 Ω，上横 12 Ω，下横 12 Ω；两条对角线导线 **交叉但不连接**（无结点）。  
- **端口**：左侧上下两点为输入端子。  
- **电流**：从端子流入后在对称桥网中分流，无独立标注电流。

**(b) 阶梯电阻网络**  
- **元件**：竖直 8 Ω；上横 3 Ω；中横 4 Ω 串 2 Ω；下横 6 Ω 串 2 Ω。  
- **连接**：左端子对 → 8 Ω 竖支；右端分三层横支，竖线连接各层节点，最右竖线汇合。  
- **电流**：从左上端子进入，经 8 Ω 及横向电阻网络，从右端或底端流出（等效电阻题，无标注方向）。

## 六、识图质量检查

- 电源是否 **全部** 标为 type=`Battery`，且符号为 **长短竖线 + ±**（不是圆圈）  
- 检流计 P 是否标为 **Galvanometer**（圆内 **一横线**，非字母 P）  
- 是否列出 **电位点** 及 **P_m —[元件]— P_n** 逐段连接  
- **开路** 是否写清：开断口、开口端子、开关断开、测量端悬空  
- 每个电流是否写 **方向 + 电位点路径**  
- 子图 (a)(b) 是否分开写  
- 半导体：VD 正偏/反偏、BJT 箭头（NPN/PNP）
